Cloud integration
=================

H2O is supported on a number of cloud environments:

- EC2 Instances and S3 Storage (RedHat AMI, Amazon Linux AMI, and Ubuntu AMI)
- Amazon AWS
- Microsoft Azure
- Databricks
- IBM DSX
- Nimbix Cloud
- Kubernetes
